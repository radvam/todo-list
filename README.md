## Or start a project locally:
* **Clone repository** 
  * $ git clone git@github.com:radvam/todo-list.git
* **Go to project folder in git-bash** 
* **Go to branch develop** 
  * $ git checkout develop  
* **Install Node modules** 
  * $ npm i 
* **Start project**
  * $ npm start
